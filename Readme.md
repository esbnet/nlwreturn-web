# FeedGet
Servidor para a aplicação FeedGet. Consiste em API RESTful para envio dos feedbacks por email e persistência das informações.

## Tecnologias

- react
- react-dom
- axios
- html2canvas
- phosphor-react
- tailwindcss
- typescript
- vite
